When do I use this?

Match your clinical scenario to the correct workflow in the app.

A

New Consult / Diagnosis

Trigger: You are seeing a patient for possible Opioid Use Disorder (OUD).

Go To Tab 4 OUD Consult
  • Toggle DSM-5 Criteria to confirm diagnosis severity.
  • Use the Note Generator to copy documentation to clipboard.
  • Run Screeners (DAST-10, ORT) if risk stratification is needed.
B

Inducing Buprenorphine

Trigger: Patient is in withdrawal or needs to start MOUD.

Go To Tab 2 Protocols
  • Select COWS (via Tab 4 or Library) to score withdrawal.
  • Select Standard for traditional induction steps.
  • Select Bernese (Micro) for cross-tapering without withdrawal.
C

Opioid Rotation / Pain

Trigger: You need to rotate opioids or calculate daily load.

Go To Tab 3 MME Calculator
  • Enter current dose to see Total Daily MME.
  • Select target opioid to see Equianalgesic Dose.
  • Note: Safety limits (Renal/Hepatic) will block unsafe options based on Tab 1 (Assessment) inputs.
